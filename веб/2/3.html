Web-терминалы

Браузер (browser)
Программа, служащая для просмотра Web-страниц
URL
Унифицированный указатель ресурсов (указывает адрес Web-документа в WWW)
Гиперссылка
Текст или графическое изображение служащее для перехода да другую страницу или метку
HTML
Язык разметки гипертекстов
CSS
Каскадные таблицы списков
PHP, Perl
Язык программирования для Web-страниц
